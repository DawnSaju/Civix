# Civix

An assistant that helps users understand the election process, timelines, and steps in an interactive and easy-to-follow way.

## Features
- **Interactive Election Timelines:** Stay up-to-date with key dates and deadlines.
- **Process Demystification:** Break down complex election rules into simple, easy-to-understand steps.
- **AI Assistant:** Ask questions and get instant, clear answers about voting requirements, registration, and more.

## Tech Stack
- Next.js
- React
